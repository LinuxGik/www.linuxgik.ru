Как работает меню на портале

Навигация на портале LinuxGIk динамическая и управляется через два основных файла в каталоге _includes/ и YAML-данные в _data/navigation.yml.

1. Точка входа (Layout)

Вся логика начинается в главном шаблоне сайта. Именно здесь подключаются скрипты, отвечающие за расчет разделов и отрисовку меню.

Файл: _layouts/default.html 
{% raw %}<body class="grid-container">

    {% include logika_header_sidebar.html %}

    <header class="site-header">
        {% include header_nav.html %}
    </header>
...{% endraw %}

2. Определение текущего раздела (logika_header_sidebar.html)

Этот файл инклудится первым в default.html. Его задача — "понять", где сейчас находится пользователь, сравнив текущий URL страницы со списком разделов в navigation.yml.

Файл: _includes/logika_header_sidebar.html 
{% raw %}{% for item in site.data.navigation.nav %}
    {% if page.url contains item.link %}
        {% assign current_section = item %}
...{% endraw %}

Суть процесса:

  • Он берет список всех разделов из site.data.navigation.nav.
  • Если в URL страницы содержится ссылка на раздел, то раздел помечается как current_section.
  • Если внутри раздела есть подразделы (subsection), он ищет совпадение в них и помечает current_subsection.

3. Вывод элементов в меню (header_nav.html)

Этот файл отвечает за визуальное отображение. В нем используется цикл for, который рисует ссылки в шапке сайта.

Файл: _includes/header_nav.html 
{% raw %}{% else %}
    {% for item in site.data.navigation.nav %}
        <li><a href="{{ item.link }}">{{ item.name }}</a></li>
    {% endfor %}
{% endif %}{% endraw %}

На данный момент этот код берет абсолютно все элементы из навигации и выводит их в шапку.

4. Зачем нам нужен "hidden"?

Именно здесь мы планируем внести изменения. Нам нужно добавить проверку: если у элемента в navigation.yml стоит метка hidden: true, то цикл в header_nav.html должен его пропустить.

При этом logika_header_sidebar.html всё равно "увидит" этот раздел, и у нас будут правильно работать хлебные крошки и сайдбары.
