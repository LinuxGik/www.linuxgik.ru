Введение

Это справочный раздел сайта www.linuxgik.ru - он предназначен для веб-мастера.

FrontMatter

В _config.yml прописан layout: default ко всем страницам в каталоге pages/ прописан в _config.yml.

Основные файлы стилей

  • assets/css/variables/ — системные и цветовые переменные.
  • assets/css/grid-layout.css — основная сетка сайта.
  • assets/css/main.css — базовые стили элементов.

Цветовая палитра

Тема использует CSS-переменные для легкой смены цветовой схемы. Ключевые цвета определены в color-var.css. _layouts/default.html
