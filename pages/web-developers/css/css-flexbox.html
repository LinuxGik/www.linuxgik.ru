CSS FlexBox

Применение к элементу свойства display: flex; превращает элемент в флекс контейнер, а потомки этого контейнера становятся его flex-элементами. По умолчанию flex-элементамы выстраиваются друг за другом в одну строку, слева на право. Flex-контейнер заполняет доступную ширину, как и блочный элемент, но flex-элементы не обязательно заполняют ширину своего flex-контейнера. Flex-элементы имеют одинаковую высоту, которая определяется контентом.
