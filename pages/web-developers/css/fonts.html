Генератор шрифтов - хороший сервис. Загружаешь шрифты в формате ttf, а он тебе форматирует их в формате woff2 и при этом даёт уже готовые css-правила для их подключения.

Как правильно подключать шрифты

Определения используемых понятий в контексте данного материала

  • Системный шрифт - шрифт который установлен в вашей операционной системе.

Для выбора шрифта используется свойство font-family. Можно использовать системные шрифты
