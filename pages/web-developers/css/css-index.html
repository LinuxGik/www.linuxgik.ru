CSS - каскадные таблице стилей. Три наиболее важные составляющие CSS: Каскадность, Относительные единицы, Блочная модель.

Каскадность

CSS - это объявленные правила. Если к одному и тому же элементу применяется несколько css-правил, это может привести к конфликту объявлений. Браузер следует конкретному набору правил и выберет одно из нескольких примененных к элементу css-правил. Каскадность - так называется этот набор браузерных-правил. Каскадность определяет способы разрешения конфликтов объявлений

Когда объявления конфликтуют, для устранения проблемы нужно учесть три показателя.

  1. Источник стилей - место их расположения. Пользовательские стили накладываются на стили браузера, применяемые по умолчанию.
  2. Специфичность селекторов - то, какие селекторы имеют приоритет над другими.
  3. Исходный порядок - порядок, в котором стили(правила) объявляются в таблице стилей

Немного терминологии

Объявление

Объявление: color: black;

Объявление состоит из свойства (color) и значения (black)

Блок объявлений

Группа объявлений внутри фигурных скобок назвается блоком объявлений.

Блоку объявлений предшествует селектор (в данном примере body):

body {
color: black;
font-family: Helvetica;
{

Набор правил

В совокупности селектор и блок объявлений назваються - набор правил.

@правила

Языковые конструкции начинающиеся с символа @, такие как запросы @import или @media.

Источники стилей

Стили перечислены в порядке возрастания приоритета. Самый маельнкий приоритет у браузерных, самый большой у важных авторских.

Браузерные стили
Стили браузера по умолчанию.
Авторские стили
Созданные вами таблицы стилей.
Важные авторские стили
Объявление может может быть помечено как важное с помощью слова !important. Пример: color: red !important;

Специфичность селекторов

Браузер оцнивает специфичность в два этапа:

  • Сначало проверяет стили, встроенные в html-код (их еще называют строчными стилями).
  • Затем стили, применяемы с помощью селекторов.

Встроенные стили

Для задания стилей конкретного элемента-html в кодt-html используется атрибут style : <h1 style="background-color: red;">Ваш заголовок</h1>

Это объявления с ограниченной областью действия, которые переопределяют любые объявления задаваемые в авторской таблице.

Специфичность селекторов

Правила специфичности:

  • Наиболее специфичный селектор с идентификтором (пример идентификатора: #page-tietle { ). Чем больше идентификаторов, тем специфичней будет селектор.
  • Селектор с классом (пример класса : .name { ), менее специфичней идентификтора. Чем больше классов, тем специфичней будет селектор.
  • Селектор с тегами (пример селктора с тегом : h1 { ), мене специфичней класса. Чем больше тегов, тем специфичней селектор.
